# Chat Auto Updater (CAU)
This is the program that automatically updates the chat mod! it's open source so PLEASE look through it if your skeptical. read below to know what it does

basically: 
MultiplayerChat downloads this if there is an update AND the "Enable CAU" setting is on.
if there is no update, no CAU will be on your computer. 
IF there is no update AND "Enable CAU" is on **AND** CAU is on your computer, MultiplayerChat deletes the CAU, then next update, downloads the latest version of CAU and then runs it.
once CAU has done its job, and automatically installed the latest version of MultiplayerChat, MultiplayerChat will delete the CAU.


the only changes that will really be made to CAU is changing the target that CAU will download (the newest release of MultiplayerChat).

thats literally ALL that CAU does. please, look through it, its a VERY basic program.

its current version target of MultiplayerChat to download: 0.3.2 .
